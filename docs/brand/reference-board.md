# Just Build with Blocks Reference Board

Date: 2026-03-05

## Shell

- Mood: OpenAI-clean, precise, quiet confidence
- Palette: white, graphite, restrained blue
- Role: navigation, inputs, instructions, exports
- Watch-out: if the shell starts feeling like a toy website, it is too loud

## Set Box

- Mood: immediate excitement
- Palette: blue hero field, red badge, yellow accent, white metadata
- Role: first magical payoff after upload or prompt
- Watch-out: avoid source-identical packaging structure or retail mimicry

## Builder Studio

- Mood: guided creative play
- Palette: white shell plus trays, boards, and primary-color pieces
- Role: the “I could make this” moment
- Watch-out: should feel like a guided 2.5D studio, not CAD software

## Build Sheet

- Mood: satisfying clarity
- Palette: white and graphite with color bins from the scene data
- Role: make the output feel real, not merely cinematic
- Watch-out: avoid faux-manual nostalgia that drifts into imitation

## Keepsake

- Mood: proud, shareable, just a little aspirational
- Palette: keep the set-box energy, but let the shell breathe
- Role: exports and `Buy the bricks` coming-soon tease
- Watch-out: commerce stays teaser-only until there is a real fulfillment story
