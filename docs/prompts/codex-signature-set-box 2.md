# Codex Signature Set Box Spec

## Objective

Create the first frame of the product: a premium original `BLOCKS` set box for the monochrome Codex collectible.

## Required content

- Original red `BLOCKS` badge
- Strong blue packaging field
- Codex Signature Set title treatment
- Monochrome hero render of the physical collectible
- Metadata rail with build age, build ID, piece count, and mode
- `Buy the bricks` teaser in a clearly non-transactional state

## Camera and composition

- Product-photography energy over toy-chaos energy
- Hero set centered and clearly legible at thumbnail scale
- The box should read as a real product without copying known retail layouts one-to-one
- Favor a calm shell around a high-excitement box face

## Anti-goals

- No counterfeit packaging layout
- No official toy-brand marks or numbering grammar
- No editorial-gallery framing
- No busy, multicolor physical scene competing with the object

## Quality bar

- First frame is screenshot-worthy
- The build looks plausibly buildable
- The wrapper feels joyful without looking noisy
- The product reads “premium collectible” before “AI demo”

## Acceptance checklist

- Does the box feel original?
- Is the monochrome Codex object unmistakable?
- Can the metadata be read quickly?
- Would someone share this first frame without extra explanation?
