# Codex Signature Instructions Spec

## Objective

Create an original instruction-book style surface that feels immediately understandable and toy-native without copying retail instruction design.

## Required content

- Pale sky-blue page field
- Large step numerals
- Parts-needed boxes per step
- Clear subassembly grouping where needed
- Directional cues and build pacing
- Final manifest / inventory section

## Layout rules

- Big whitespace
- High legibility at export size
- Step order derived from canonical assemblies and phases
- Manifest totals must match the real model

## Anti-goals

- No copied warning layouts or branded covers
- No dense technical spreadsheet feeling
- No detached instruction order that disagrees with the studio

## Quality bar

- Someone could plausibly follow the steps
- The page feels premium and calm
- The instruction system looks like part of the same product family as the box and studio

## Acceptance checklist

- Do steps map to assemblies cleanly?
- Are parts-needed callouts readable?
- Would the instructions still make sense in print?
- Does the final inventory feel trustworthy?
