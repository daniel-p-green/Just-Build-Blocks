# Codex Signature Studio Spec

## Objective

Create a guided 3D studio that helps someone inspect and understand the real Codex collectible without turning into CAD.

## Required controls

- Orbit camera
- Build phase stepper
- Explode / assemble toggle
- Instruction sync toggle
- Play reveal action

## Scene language

- OpenAI-clean shell
- Product-demo stage, not sandbox editor
- Premium board/plinth presentation
- Parts should feel like collectible set pieces, not raw voxel blocks

## Anti-goals

- No brick-by-brick dragging
- No generic editor chrome
- No overcomplicated camera controls
- No confusing mismatch between instruction phases and visible assemblies

## Quality bar

- The model feels physical and legible
- The build phases are easy to follow
- The studio feels playful, but disciplined
- The studio proves the canonical model instead of distracting from it

## Acceptance checklist

- Can a first-time viewer tell what they are looking at?
- Do the build phases reveal the set coherently?
- Does explode view help rather than confuse?
- Does the studio feel closer to a toy companion than to a modeling tool?
